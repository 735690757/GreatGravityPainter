# GreatGravityPainter

基于浏览器的地球物理可视化工具箱，支持三维地质体渲染与二维重力场绘制。无需安装任何依赖，使用现代浏览器直接打开即可运行。

## 功能特性

### 三维地质体可视化

- 加载 20x20x20 体素密度数据（`.txt` 格式）
- 体素渲染，可调整整体透明度与密度阈值过滤
- 沿 X / Y / Z 轴的交互式剖面切割，附带 2D 剖面小窗
- 多种配色方案（Jet、Viridis、Hot、Cool）
- 真实距离模式（单位：米）
- 鼠标悬浮体素显示 XYZ 坐标与密度值
- 鼠标左键拖拽旋转 / 滚轮缩放 / 右键平移

### 二维重力场绘制

- 加载 41x41 重力观测数据（x, y, z, gravity 四列）
- 色彩映射热力图 + 等值线叠加
- 多种插值方式与配色方案
- 鼠标悬浮查看数据值

### 通用功能

- 支持拖拽文件加载或文件夹选择（File System Access API）
- 纯前端运行，无需后端服务，数据不离开本地
- 动态星空背景首页，鼠标交互光效

## 快速开始

1. 克隆仓库：
   ```bash
   git clone https://github.com/KarryLiu/GreatGravityPainter.git
   ```
2. 用现代浏览器打开 `index.html`（推荐 Chrome / Edge 以获得完整的文件系统访问支持）。
3. 选择可视化模式，加载你的数据文件即可。

## 数据格式

### 三维密度数据

纯文本文件，每行一个数值，共 8000 行（20 x 20 x 20 网格）。数值表示密度，范围 [-1, +1]。

### 二维重力数据

纯文本文件，每行 4 列（x y z gravity），共 1681 行（41 x 41 网格）。

## 技术栈

- [Three.js](https://threejs.org/) — 三维渲染与交互
- 原生 HTML / CSS / JavaScript — 零构建步骤
- File System Access API — 原生文件夹/文件选取

## 浏览器兼容性

完整功能需要基于 Chromium 的现代浏览器（Chrome、Edge）。拖拽加载功能在所有现代浏览器中可用。
